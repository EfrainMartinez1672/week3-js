# 📝 Mi Lista de Notas (Mini-App)

Una aplicación web minimalista y elegante para gestionar notas personales. Este proyecto fue desarrollado como una práctica práctica de **manipulación del DOM** y persistencia de datos del lado del cliente mediante **Local Storage**, todo envuelto en un diseño moderno con una paleta de colores morados oscuros.

---

## 🚀 Características

* **Persistencia Total:** Las notas se guardan automáticamente en el navegador. Si recargas la página, tus notas seguirán ahí.
* **Interactividad en Tiempo Real:** Agrega y elimina elementos del DOM de forma dinámica sin necesidad de refrescar la página.
* **Validación de Datos:** El sistema evita el ingreso de campos vacíos alertando al usuario.
* **Diseño Confortable:** Interfaz adaptada con un tema *Dark Purple* que reduce la fatiga visual.

---
